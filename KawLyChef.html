⚙️ Configuration Firebase

Étape 1 — Créer votre projet (gratuit)

1. Allez sur console.firebase.google.com
2. "Ajouter un projet" → nom : kawlychef → Créer

Étape 2 — Activer Google Sign-In

Authentication → Sign-in method → Google → Activer → Enregistrer

Étape 3 — Activer Firestore

Firestore Database → Créer → Mode test → Région : europe-west3

Étape 4 — Activer Storage

Storage → Commencer → Mode test

Étape 5 — Récupérer votre config

Paramètres ⚙️ → Paramètres du projet → Vos applications → icône Web </>
Copiez les 6 valeurs firebaseConfig

Étape 6 — ✅ Config déjà intégrée !

Votre config Firebase kaw-ly-chef est déjà dans l'application.
Il reste juste à activer votre domaine GitHub Pages dans Firebase.

✅ Dernière étape pour GitHub Pages

Dans Firebase Console → Authentication → Domaines autorisés
Ajoutez : VOTRE_USERNAME.github.io
Exemple : kawly.github.io
